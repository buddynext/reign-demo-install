One-click demo installer for Reign WordPress Theme with complete user session preservation.

== Description ==

Reign Demo Install allows you to import pre-built demo sites for the Reign WordPress theme with just one click. The plugin ensures your admin account is preserved throughout the import process, so you never lose access to your site.

= Key Features =

* One-click demo import
* Admin user preservation - stay logged in during import
* Automatic backup before import
* Support for BuddyPress/BuddyBoss content
* Premium plugin installation support
* Media and file import
* Settings and customization import
* Rollback capability with backup restoration

= User Preservation =

The plugin's unique user preservation system ensures:
* Your admin account is never deleted or modified
* You remain logged in throughout the import process
* Your login credentials are preserved
* Session management prevents unexpected logouts

= Demo Content =

Import complete demo sites including:
* Posts, pages, and custom post types
* Menus and menu locations
* Widgets and widget areas
* Theme customizations
* Plugin settings
* Media files and uploads
* BuddyPress/BuddyBoss data

= Requirements =

* WordPress 6.0 or higher
* PHP 7.4 or higher
* Reign theme active
* 256MB memory limit (512MB recommended)
* 500MB free disk space

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/reign-demo-install` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure Reign theme is active
4. Navigate to Appearance > Install Demos or Reign Demos in the admin menu
5. Browse available demos and click Import on your chosen demo
6. Select your import options and click Start Import

== Frequently Asked Questions ==

= Will I lose access to my site during import? =

No. The plugin preserves your admin account and keeps you logged in throughout the import process.

= Can I undo a demo import? =

Yes. The plugin creates automatic backups before import that can be restored anytime.

= Do I need license keys for premium plugins? =

Yes. If a demo uses premium plugins, you'll need to enter your license keys in the plugin settings.

= What happens to my existing content? =

By default, existing content is preserved. You can choose "Clean Install" to remove existing content (except your admin account).

= How long does import take? =

Import time varies based on demo size and server performance. Most demos import within 2-5 minutes.

= Can I import just parts of a demo? =

Yes. You can choose which components to import: content, media, users, settings, etc.

== Demo Import Process ==

1. **Browse Demos**: View all available demos with previews
2. **Select Options**: Choose what to import (content, media, settings, etc.)
3. **Preserve User**: Your admin account is automatically preserved
4. **Create Backup**: Optional automatic backup before import
5. **Import Content**: Demo content is imported step by step
6. **Stay Logged In**: Session management keeps you logged in
7. **Complete**: Redirects to your new demo site

== Technical Details ==

The plugin uses a sophisticated import system:

* **User Preservation**: Stores complete user data and session information
* **Step-by-Step Import**: Processes import in manageable chunks
* **Session Management**: Prevents logouts with cookie management
* **Error Recovery**: Automatic retry and session restoration
* **Progress Tracking**: Real-time import progress updates
